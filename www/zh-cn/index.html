Pyarmor

一个加密Python源代码的命令行工具

了解更多

无缝替换

经过Pyarmor加密后的脚本也是一个合法的Python脚本。只需要额外的一个模块 pytransform.py 和几个辅助文件，加密后的脚本可以无缝的替换的原来的脚本，然后被Python解释器正常导入和运行。

设置运行期限

可以设置加密脚本的运行期限，过期之后加密脚本就不能被运行

绑定到硬盘或者网卡

可以绑定加密后的脚本到硬盘序列号，网卡地址等，限制加密脚本只能在某一台机器上运行

多平台支持
  • Python 2.5, 2.6, 2.7 和 Python3
  • win32, win_amd64, linux_i386, linux_x86_64, macosx_intel
  • 嵌入式平台: Raspberry Pi, Banana Pi, TS-4600 / TS-7600

快速开始

  • 安装
    pip install pyarmor
  • 加密脚本
    python pyarmor.py obfuscate --src=examples/simple --entry=queens.py
  • 运行加密脚本
    cd dist
    python queens.py
