A command line tool used to obfuscate python scripts.

Learn More Quick Start

Seamless Replacement

The obfuscated script is a normal python script with an extra module pytransform.py and a few extra runtime files. So original python scripts can be replaced with obfuscated scripts seamlessly.

And works well with py2exe, py2app, PyInstaller etc.

Runtime Obfuscation

The byte code of each code object will be obfuscated as soon as code object completed execution, and f_locals of frame is cleared at the same time.

Different Licenses

Expire obfuscated scripts

Bind obfuscated scripts to serial number of harddisk, mac address of network card, ip address ect.

Support Platforms

Python 2.5, 2.6, 2.7 and Python3

win32, win_amd64, linux_i386, linux_x86_64, macosx_intel

Quick Start

  • Install
    
  pip install pyarmor
  • Obfuscate Scripts
    
  cd /path/to/pyaror
  python pyarmor.py obfuscate --src=examples/simple --entry=queens.py
  • Run Obfuscated Scripts
    
  cd ./dist
  python queens.py
  • Bind to fixed machine and expire obfuscated scripts
    
  # First get hardware information
  python pyarmor.py hdinfo

  # Sample output
  Serial number of first harddisk: '100304PBN2081SF3NJ5T'
  Mac address: '70:f1:a1:23:f0:94'
  Ip address: '192.168.121.101'
  Domain name: ''
          
        
    
  # Generate new license
  python pyarmor.py licenses --expired 2018-12-31 \
                             --bind-disk "100304PBN2081SF3NJ5T" \
                             --bind-mac "70:f1:a1:23:f0:94" \
                             --bind-ipv4 "202.10.2.52" \
                             customer-jondy

  # Overwrite default license
  cp licenses/customer-jondy/license.lic ./dist

  # Run obfuscated scripts under new license
  cd dist
  python queens.py
