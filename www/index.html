Pyarmor

A command line tool used to obfuscate python scripts, bind obfuscated scripts to fixed machine or expire obfuscated scripts.

Learn More Quick Start

Seamless Replacement

The obfuscated script is a normal python script. With an extra module pytransform.py and a few extra runtime files, clear Python scripts can be replaced with obfuscated ones seamlessly

And Pyarmor works well with py2exe, py2app, PyInstaller, cx_Freeze etc.

Runtime Obfuscation

The byte code of each code object will be obfuscated as soon as code object completed execution, and f_locals of frame is cleared at the same time

It can protected those Python source files distributed as package

Pack Obfuscated Scripts

Pyarmor could pack obfuscated scripts with py2exe, py2app, PyInstaller, cx_Freeze etc.

Different Licenses

Expire obfuscated scripts

Pyarmor can set expired date for obfuscated scripts. After that, obfuscated scripts can not be run

Bind to fixed machine

Pyarmor can bind obfuscated scripts to serial number of harddisk, mac address of network card, ip address etc..

Quick Start

Install Pyarmor

The easy way is by pip, or download source package from pypi/pyarmor

pip install pyarmor
Obfuscate Scripts

Obfuscate all the .py files in the path --src by command obfuscate

cd /path/to/pyaror
python pyarmor.py obfuscate --src=examples/simple --entry=queens.py
Run Obfuscated Scripts

The obfuscated scripts and all the required runtime files are generated in the default output path dist

Run obfuscated script queens.py as normal Python script

cd ./dist
python queens.py
Pack Obfuscated Scripts

Pack obfuscated scripts by command pack

python pyarmor.py pack -t cx_Freeze examples/py2exe/hello.py
Generate Different Licenses

First get hardware information by command hdinfo, it will print hardware information in console window

Then generate a new license with command licenses, new license file will be saved as licenses/CODE/license.lic

Next overwrite default license dist/license.lic

The default license is generated while obfuscating scripts, it allows obfuscated scripts run in any machine and never expired

Finally run obfuscated script queens.py with new license

python pyarmor.py hdinfo
python pyarmor.py licenses \
                --expired "2018-12-31" \
                --bind-disk "100304PBN2081SF3NJ5T" \
                --bind-mac "70:f1:a1:23:f0:94" \
                --bind-ipv4 "202.10.2.52" \
                customer-jondy

cp licenses/customer-jondy/license.lic ./dist

cd dist
python queens.py
WebUI

Run pyarmor-webui to start web page. Only for basic usage, full features still need shell command way

It will start a light weight web server in localhost, and open a page in web browser.

pyarmor-webui
