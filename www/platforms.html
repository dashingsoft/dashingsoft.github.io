PyArmor Prebuilt Dynamic Libraries

The core of PyArmor is written by C, the prebuilt dynamic libraries include the common platforms and some embeded platforms.

Some of them are distributed with PyArmor package, refer to Table-1.

Some of them are not, refer to Table-2. In these platforms, in order to run pyarmor, first download the corresponding prebuilt dynamic library, then put it in the installed path of PyArmor package.

Contact jondy.zhao@gmail.com if you'd like to run PyArmor in other platform.

Table 1. Prebuilt Libraries Distributed with PyArmor
# OS Arch Features Download Remark

  • AntiDebug
  • JIT
 _pytransform.dll Cross compile by i686-pc-mingw32-gcc in cygwin
Windows AMD64
  • AntiDebug
  • JIT
 _pytransform.dll Cross compile by x86_64-w64-mingw32-gcc in cygwin
Linux i686
  • AntiDebug
  • JIT
 _pytransform.so Built by GCC
Linux x86_64
  • AntiDebug
  • JIT
 _pytransform.so Built by GCC
Linux armv7 _pytransform.so Cross compile by GCC in ubuntu 16.04 (x86_64)
for 32-bit Armv7 Cortex-A, hard-float, little-endian
Linux aarch64 _pytransform.so Cross compile by GCC in ubuntu 16.04 (x86_64)
for 64-bit Armv8 Cortex-A, little-endian
MacOSX x86_64, intel
  • AntiDebug
  • JIT
 _pytransform.dylib Built by CLang with MacOSX10.11
